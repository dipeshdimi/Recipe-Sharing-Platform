### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/dipeshdimi/Recipe-Sharing-Platform.git
    ```

2. **Setup Backend:**

    ```bash
    cd server
    npm install
    npm start
    ```

3. **Setup Frontend:**

    ```bash
    cd client
    npm install
    npm run dev
    ```
